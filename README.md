# Коррекция фото в браузере

Небольшое веб-приложение для автоматической коррекции яркости, контраста и насыщенности изображения. Все основные вычисления выполняются в браузере пользователя.

После автоматической обработки пользователь видит параметры, предложенные моделью, и может изменить их вручную с помощью ползунков.

## Запуск

В папке проекта выполните:

```bash
python -m http.server 8000
```

Затем откройте:

```text
http://localhost:8000
```
